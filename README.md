# Smart Parking Management System (Java)

A console-based Smart Parking Management System built using Core Java and Object-Oriented Programming principles.  
The system allows users to park vehicles, remove vehicles, search for vehicles, and track parking slot availability.

---

## Features

- Park vehicles in the nearest available slot
- Remove vehicles from parking slots
- Calculate parking fees based on parking duration
- Search for a vehicle by vehicle number
- Display parking slot status
- Menu-driven interactive console application

---

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList (Java Collections)
- Scanner for user input

## How It Works

1. User enters number of parking slots.
2. User can park vehicles by entering vehicle number and type.
3. System automatically assigns the first available slot.
4. Parking fee is calculated when the vehicle exits.
5. Users can search for a vehicle by vehicle number.
6. Parking status shows which slots are occupied.
